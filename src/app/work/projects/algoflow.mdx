## Overview

AlgoFlow is an interactive web app built to help learners understand how sorting algorithms work by visualizing them in real-time. The idea was to create a clean, intuitive, and engaging interface where users can adjust parameters and watch how the algorithms manipulate data step by step.

## Key Features

- **Algorithm Visualizations**: Implemented animations for Bubble Sort, Insertion Sort, Selection Sort, Merge Sort, and Quick Sort to show how each algorithm processes data.
- **Custom Controls**: Users can change array size, adjust animation speed, and step through the sort manually to get a deeper understanding of the logic.
- **Performance Tracking**: Display of execution times for each sorting algorithm, encouraging comparative learning and awareness of complexity.
- **Responsive UI**: Designed for use across devices, from desktops to mobile, ensuring accessibility for students and curious learners anywhere.

## Technologies Used

- **React w/Next.js**: For building the interactive UI and managing state transitions between algorithm steps.
- **Tailwind CSS**: For fast, utility-first styling and responsive layout design.
- **Vercel**: For seamless deployment and fast global delivery.

## Challenges and Learnings

The biggest challenge was animating algorithm steps in a way that remained faithful to the logic but still felt smooth and intuitive. Managing state updates between sorting steps without blocking the UI required careful handling of async timing and React state. I also focused on making the app accessible and lightweight.

## Outcome

AlgoFlow makes sorting algorithms tangible and easier to understand by combining theory with visual interaction. It’s currently used by students and self-taught developers to explore how different algorithms behave under various conditions — and has helped make algorithm learning more engaging and approachable.
