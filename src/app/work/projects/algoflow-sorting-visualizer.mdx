## Overview

AlgoFlow is an interactive web application designed to demystify sorting algorithms for learners. By visualizing algorithms like Bubble Sort, Merge Sort, and Quick Sort in real-time, it provides an engaging way to understand their mechanics and performance. The app combines a sleek interface with intuitive controls to make algorithm learning accessible to students and developers alike.

## Technologies Used

AlgoFlow leverages modern web technologies to deliver a smooth and efficient experience:

- **React with Next.js**: Powers the interactive UI and manages state for smooth transitions between algorithm steps.
- **Tailwind CSS**: Enables rapid, utility-first styling and responsive layouts.
- **Vercel**: Ensures fast deployment and global accessibility.

## Challenges and Learnings

Animating algorithm steps without compromising accuracy or performance was a key challenge. Managing asynchronous state updates in React to ensure smooth animations required careful optimization. Additionally, ensuring accessibility across devices meant rigorous testing and responsive design adjustments.
